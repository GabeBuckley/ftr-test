Help

Welcome to fibbr, an app that determines if an entered number is one of the first 1000 Fibonacci numbers.

Initially you will be asked to set a timer interval. This interval governs the length of time between display updates. You can halt and resume the timer at any point during the demonstration.

From there, you will be asked to enter a sequence of positive whole integers (numbers). If you enter a number that is part of the Fibonacci sequence, the word FIB will appear. If the number you entered is not part of the Fibonacci sequence, then the word FIB will not appear.

The numbers you have entered into the system will be displayed on screen according to the interval you set up at the start.
